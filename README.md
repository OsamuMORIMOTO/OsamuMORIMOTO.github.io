# Installation

```
$ git clone git@github.com:OsamuMORIMOTO/blog.git
$ npm install
```

# Usage

## make draft

```
$ npm run draft [title]
```

## publish draft

```
$ npm run publish [title]
```

## up preview server & watch file

```
$ npm start
```

## deploy to github

```
$ npm run clean-deploy
```

or

```
$ npm run c-d
```

# recommend setting

```
$ npm completion >> ~/.bashrc
$ source ~/.bashrc
```
